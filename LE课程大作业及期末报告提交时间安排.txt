第一部分    编程实践项目 (30%)

在以下任务中任选一个任务完成

project #1  (30%)

完成一个语料库管理程序，功能包括：

（1）提供对不同编码的txt文件（包括GBK字符集、Unicode、UTF-8字符集）的支持；
（2）提供语料文件的基本管理功能，如添加、删除等。
（3）提供串频统计（N-gram）功能：可产生语料库中的字串、词串频度文件（两种排序：字母序|频次序），用户可指定 N 值范围，字串频次范围等；
（4）提供关键字检索功能；
（5）提供不连续关键字检索功能：
   例如： 输入查询A <2,10> B，其中A，B为字符串，<2,10>表示A与B之间间隔2到10个字符，返回所有符合查询条件的句子。
（6）可以支持在语料中查询常见搭配词、相似词、相似例句。

project #2  (30%)

编写一个词典知识库查询程序，对一个知识库（比如：access格式的汉语语义词典数据库，构式知识库，XML格式的propbank、verbnet词库等）进行较为全面的分类统计，对词典的知识描述质量做出评估。

功能示例：（1） 词典条目的分类、各类数量比例、排序； （2）语义分类树节点对应的词例数量； （3）词典内部一致性

可参考：http://ccl.pku.edu.cn/ccgd/stat.asp 对构式知识库进行统计的网页，并做出改进。

第三部分    面向信息处理的语言学专题研究  (40%)

在以下任务中选一个任务完成

project #1   构式与短语结构句法分析的融合 (40%)
（1）收集一定数量的含有构式的真实语料句子。对这些句子进行自动句法结构分析，并对自动分析结果进行人工校对，提交句法结构信息标注正确的结果文件。

（2）在短语结构分析规则库中添加构式分析规则，使得计算机可以针对包含构式的句子做更有效的句法分析处理，分析添加了构式相关分析规则后自动句法分析结 果改进的情况，

（3）对构式分析规则与短语结构分析规则融合涉及的语言学问题和计算机处理问题进行讨论，撰写报告。

project #2   包含特定词语的真实例句的语义分析 (40%)

（1）选取一些用法和意义多样的动词（比如“拿、取、得”等，可以在现有语义词库中查找义项较多，意义相近的条目），收集包含它们的真实语料句子，对这些句子进行语义分析（比如谓词语义角色标注），提交正确 的标注结果。

（2）对语义分析中碰到的问题进行归纳，讨论，提出解决方案，撰写报告。
